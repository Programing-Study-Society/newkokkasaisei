Unity バージョン 2022.3.3f1

・パスカルケース
　最初大文字 + 大文字区切り
　例：UserName

・キャメルケース
　最初小文字 + 大文字区切り
　例：userName

・スネークケース
　全小文字 ＋ アンダースコア区切り
　例：user_name

・アッパースネークケース
　全大文字 ＋ アンダースコア区切り
　例：USER_NAME

・ハンガリアン記法
　接頭辞 ＋ 名前
　txtUserName

・チェインケース
　全文字小文字 ＋ ハイフン区切り
　例：user-name
